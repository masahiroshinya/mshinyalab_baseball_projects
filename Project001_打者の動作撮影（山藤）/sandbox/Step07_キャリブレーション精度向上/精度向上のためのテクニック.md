# Step 07：精度向上のためのテクニック（1：キャリブレーション編）

3次元復元の精度を左右する最大の要因は**キャリブレーションの質**です。再投影誤差（RMS）が大きく、計測値に不安がある場合は、以下の手順で改善を試みます。

## 1. 再投影誤差（RMS）の詳細分析

全体の平均誤差（RMS）を見るだけでなく、「どの画像が悪影響を与えているか」を特定します。

### 個別画像の誤差計算コード

以下のコードをColabのキャリブレーション直後に実行してください。

```python
# 各画像ごとの誤差を計算する関数
def analyze_reprojection_errors(objpoints, imgpoints, rvecs, tvecs, K, D):
    total_error = 0
    errors = []
  
    for i in range(len(objpoints)):
        imgpoints2, _ = cv2.projectPoints(objpoints[i], rvecs[i], tvecs[i], K, D)
        error = cv2.norm(imgpoints[i], imgpoints2, cv2.NORM_L2) / len(imgpoints2)
        errors.append(error)
        total_error += error
        print(f"Image {i}: Error = {error:.4f} px")
  
    mean_error = total_error / len(objpoints)
    print(f"\nMean error: {mean_error:.4f} px")
    return errors

# 実行
errors = analyze_reprojection_errors(objpoints1, imgpoints1, rvecs1, tvecs1, K1, D1)

# 誤差が大きい画像（例：1.0px以上）のインデックスを特定
bad_indices = [i for i, error in enumerate(errors) if error > 1.0]
print(f"推奨削除対象インデックス: {bad_indices}")
```

## 2. 悪いデータの除外と再計算

特定の画像でコーナー検出が甘かったり、チェスボードが歪んで写っていたりすると、全体の精度が下がります。

1. **目視確認**: 上記で `bad_indices` に挙がった画像を `cv2_imshow` で確認します。コーナーがズレていないかチェックしてください。
2. **データのフィルタリング**: 誤差の大きい画像を除外して、再度 `cv2.calibrateCamera` または `cv2.stereoCalibrate` を実行します。
3. **RMSの改善を確認**: 0.5px以下を目指します。

## 3. チェスボード撮影の「カバレッジ」向上

良好なキャリブレーションには、以下の条件を満たす20枚程度の画像ペアが必要です。

- **画面全体を網羅**: 中央だけでなく、四隅（レンズの歪みが大きい場所）にもチェスボードを配置してください。
- **角度の変化**: 正面だけでなく、上下左右に30度程度傾けた状態も混ぜます。
- **近距離と遠距離**: カメラに近い位置と、実際に動作を計測する距離の両方で撮影します。

## 4. サブピクセル精度の徹底

コーナー検出時に `cv2.cornerSubPix` を使用していることを確認してください（Step 05の実装ステップに記載済み）。これを行わないと、ピクセル単位の誤差（1.0px程度）が避けられません。
