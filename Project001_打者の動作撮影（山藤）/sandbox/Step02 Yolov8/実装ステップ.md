## ステップ 4：モデルファイルの準備

YOLOv8-Pose には複数のサイズがある。まずは `yolov8n-pose`（最小・最速）で試す。

| モデル           | ファイルサイズ | 精度 (AP) | 処理速度 | 用途             |
| ---------------- | -------------- | --------- | -------- | ---------------- |
| `yolov8n-pose` | ~6 MB          | ~50%      | 最速     | 動作確認・学習用 |
| `yolov8s-pose` | ~23 MB         | ~60%      | 速い     | バランス型       |
| `yolov8m-pose` | ~52 MB         | ~65%      | 普通     | 精度重視         |
| `yolov8l-pose` | ~87 MB         | ~68%      | 遅い     | 高精度           |
| `yolov8x-pose` | ~137 MB        | ~69%      | 最遅     | 最高精度         |

### モデルのダウンロード方法

スクリプト内に書けば**自動でダウンロード**される（初回実行時のみ）。
手動でダウンロードしたい場合は [Ultralytics GitHub Releases](https://github.com/ultralytics/assets/releases) から取得できる。

## ステップ 7：キーポイントの数値データをCSVに保存

検出された17点の座標（x, y）と信頼度スコアをCSVに書き出す。

```python
# save_keypoints.py
from ultralytics import YOLO
import pandas as pd

# COCO形式の17点キーポイント名
KEYPOINT_NAMES = [
    "nose", "left_eye", "right_eye", "left_ear", "right_ear",
    "left_shoulder", "right_shoulder", "left_elbow", "right_elbow",
    "left_wrist", "right_wrist", "left_hip", "right_hip",
    "left_knee", "right_knee", "left_ankle", "right_ankle"
]

IMAGE_PATH = "/content/img1.jpg"
MODEL_NAME = "yolov8n-pose"
OUTPUT_CSV = "/content/keypoints.csv"

model = YOLO(f"{MODEL_NAME}.pt")
results = model(IMAGE_PATH)

all_rows = []  # 全人物分をまとめるリスト

for result in results:
    if result.keypoints is None:
        print("キーポイントが検出されませんでした")
        continue

    kps_np = result.keypoints.data.cpu().numpy()  # shape: (人数, 17, 3) — x, y, conf

    for person_idx, person_kps in enumerate(kps_np):
        for i, name in enumerate(KEYPOINT_NAMES):
            x, y, conf = person_kps[i]
            all_rows.append({
                "person_id": person_idx,
                "keypoint": name,
                "x": round(float(x), 2),
                "y": round(float(y), 2),
                "confidence": round(float(conf), 4)
            })

df = pd.DataFrame(all_rows)
df.to_csv(OUTPUT_CSV, index=False)  # ループの外で1回だけ保存
print(f"CSVを保存しました: {OUTPUT_CSV}")
print(df.to_string(index=False))
```

### 実行

新しいセルに貼り付けて実行する。

### 出力例（CSV）

```

0,nose,320.5,112.3,0.9876
0,left_eye,305.2,100.1,0.9512
0,right_eye,335.8,99.8,0.9488
...
```
