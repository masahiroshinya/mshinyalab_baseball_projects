## ステップ 4：物体検出モデルの実行

Step02 では姿勢推定モデル（`yolov8n-pose`）を使ったが、
ここでは**物体検出モデル**（`yolov8n`）を使う。

```python
from ultralytics import YOLO

IMAGE_PATH = "/content/img1.jpg"
MODEL_NAME = "yolov8n"  # 物体検出モデル（pose なし）

model = YOLO(f"{MODEL_NAME}.pt")
results = model(IMAGE_PATH)

print("推論完了！")
```

### 期待される出力

```
image 1/1 /content/img1.jpg: 480x640 x persons, x baseball bats, xx.xms
推論完了！
```

### エラーが出た場合

| エラーメッセージ                                       | 原因               | 対処法                          |
| ------------------------------------------------------ | ------------------ | ------------------------------- |
| `ModuleNotFoundError: No module named 'ultralytics'` | インストール未完了 | ステップ1を再実行               |
| `FileNotFoundError: /content/img1.jpg`               | 画像パスが違う     | ステップ3のアップロードを再実行 |
| バットが検出されない                                   | 精度不足           | モデルサイズを大きくする（n→s→m） |

## ステップ 6：バットのバウンディングボックスをCSVに保存

COCO クラス番号 38 (`baseball bat`) のボックスだけを抽出して保存する。

```python
import pandas as pd

OUTPUT_CSV = "/content/bat_boxes.csv"
BAT_CLASS_ID = 34  # baseball bat の COCOクラス番号（38はtennis racket）

all_rows = []

for result in results:
    boxes = result.boxes
    for i in range(len(boxes)):
        cls_id = int(boxes.cls[i].item())
        if cls_id != BAT_CLASS_ID:
            continue  # バット以外はスキップ
        x1, y1, x2, y2 = boxes.xyxy[i].tolist()
        conf = float(boxes.conf[i].item())
        all_rows.append({
            "class": "baseball bat",
            "x1": round(x1, 2),
            "y1": round(y1, 2),
            "x2": round(x2, 2),
            "y2": round(y2, 2),
            "confidence": round(conf, 4)
        })

if all_rows:
    df = pd.DataFrame(all_rows)
    df.to_csv(OUTPUT_CSV, index=False)
    print(f"CSVを保存しました: {OUTPUT_CSV}")
    print(df.to_string(index=False))
else:
    print("バットが検出されませんでした。モデルサイズを大きくして再試行してください。")
```

### 出力例（CSV）

```
class,x1,y1,x2,y2,confidence
baseball bat,520.3,180.5,640.0,250.2,0.8732
```
