# Step05：2台のカメラでの撮影とキャリブレーション

## 概要

2台のカメラで同じ被写体を撮影し、三次元座標を取得するための技術ステップ。

```
[各カメラ単体キャリブレーション]
       ↓
[ステレオキャリブレーション]
       ↓
[ステレオ平行化（Rectification）]
       ↓
[対応点の検出]
       ↓
[三角測量（Triangulation）→ 3D座標取得]
```

---

## ステップ1：チェスボードの準備とパラメータ定義

チェスボードは「内側コーナーの数」で定義する。

| パラメータ | 意味 | 例 |
|---|---|---|
| `pattern_size = (cols, rows)` | 内側コーナーの列数×行数 | (9, 6) |
| `square_size` | 1マスの実寸（mm） | 25.0 mm |

> **注意：** 内側コーナーとは白黒の境界点のこと。10×7マスのボードなら `(9, 6)`。

```python
pattern_size = (9, 6)   # チェスボードの内側コーナー数（列, 行）
square_size = 25.0       # 1マスの実寸 [mm]

# 世界座標の生成（Z=0の平面上にコーナーが並ぶと仮定）
objp = np.zeros((pattern_size[0] * pattern_size[1], 3), np.float32)
objp[:, :2] = np.mgrid[0:pattern_size[0], 0:pattern_size[1]].T.reshape(-1, 2)
objp *= square_size
```

---

## ステップ2：コーナー検出（各画像に対して実行）

```python
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
ret, corners = cv2.findChessboardCorners(gray, pattern_size, None)

if ret:
    # サブピクセル精度に精密化
    criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 30, 0.001)
    corners = cv2.cornerSubPix(gray, corners, (11, 11), (-1, -1), criteria)

    # 可視化確認
    cv2.drawChessboardCorners(img, pattern_size, corners, ret)
```

出力される `corners` の形状：`(N, 1, 2)` ← N個のコーナー座標 (x, y)

---

## ステップ3：単体カメラキャリブレーション（内部パラメータ推定）

各カメラについて個別に実行する。

```python
ret, K, D, rvecs, tvecs = cv2.calibrateCamera(
    objpoints,   # 世界座標（3D）のリスト：各画像分
    imgpoints,   # 画像座標（2D）のリスト：各画像分
    image_size,  # 画像サイズ (width, height)
    None, None
)
print(f"再投影誤差 (RMS): {ret:.4f} px")  # 0.5〜1.0以下が目標
```

**出力パラメータ：**

| パラメータ | 行列形状 | 意味 |
|---|---|---|
| `K`（カメラ行列） | 3×3 | 焦点距離 (fx, fy)、主点 (cx, cy) |
| `D`（歪み係数） | 5×1 | 樽型・糸巻型歪み、接線歪み |
| `rvecs` | 各画像の回転ベクトル | チェスボードの向き |
| `tvecs` | 各画像の並進ベクトル | チェスボードの位置 |

カメラ行列の構造：

```
K = [[fx,  0, cx],
     [ 0, fy, cy],
     [ 0,  0,  1]]
```

再投影誤差（RMS）が **0.5〜1.0ピクセル以下** なら良好なキャリブレーション。

## ステップ4：ステレオキャリブレーション（外部パラメータ推定）

2台のカメラの相対的な位置・向きを求める。

```python
ret, K1, D1, K2, D2, R, T, E, F = cv2.stereoCalibrate(
    objpoints,    # 共通の世界座標
    imgpoints1,   # カメラ1の画像座標リスト
    imgpoints2,   # カメラ2の画像座標リスト
    K1, D1, K2, D2,
    image_size,
    flags=cv2.CALIB_FIX_INTRINSIC  # 単体キャリブ済みなら固定
)
```

**出力パラメータ：**

| パラメータ | 形状 | 意味 |
|---|---|---|
| `R` | 3×3 | カメラ1→カメラ2の回転行列 |
| `T` | 3×1 | カメラ間の並進ベクトル（実寸mm単位） |
| `E` | 3×3 | エッセンシャル行列 |
| `F` | 3×3 | ファンダメンタル行列 |

---

## ステップ5：ステレオ平行化（Rectification）

2枚の画像を、対応点が同じ行（エピポーラ線が水平）になるよう変換する。

```python
R1, R2, P1, P2, Q, roi1, roi2 = cv2.stereoRectify(
    K1, D1, K2, D2, image_size, R, T
)

# 各ピクセルのリマップテーブル作成
map1x, map1y = cv2.initUndistortRectifyMap(K1, D1, R1, P1, image_size, cv2.CV_32FC1)
map2x, map2y = cv2.initUndistortRectifyMap(K2, D2, R2, P2, image_size, cv2.CV_32FC1)

# 画像を平行化
rect1 = cv2.remap(img1, map1x, map1y, cv2.INTER_LINEAR)
rect2 = cv2.remap(img2, map2x, map2y, cv2.INTER_LINEAR)
```

平行化後は、同じy座標にある点が対応点になるため、後続処理が容易になる。

---

## ステップ6：対応点の検出

平行化済み画像ペアから、同じ特徴点を両画像で特定する。方法は2種類。

### A. 特定点（手首・バットなど）を指定

Step02〜04で検出したYOLO結果をそのまま使用する場合：

```python
# 各カメラ画像でYOLO推論 → 対応する点を記録
pts1 = np.array([[x1_cam1, y1_cam1], ...], dtype=np.float32)
pts2 = np.array([[x1_cam2, y1_cam2], ...], dtype=np.float32)
```

### B. 視差マップ（全領域の奥行きを一括取得）

```python
stereo = cv2.StereoSGBM_create(
    minDisparity=0, numDisparities=128, blockSize=11
)
disparity = stereo.compute(rect1_gray, rect2_gray)
```

## ステップ7：三角測量（Triangulation → 3D座標）

対応する2D点から3D座標を計算する。

```python
# 点の形式: (2, N) float32
pts1_T = pts1.T.astype(np.float32)
pts2_T = pts2.T.astype(np.float32)

# 三角測量
points4D = cv2.triangulatePoints(P1, P2, pts1_T, pts2_T)

# 同次座標 → 3D座標に変換
points3D = (points4D[:3] / points4D[3]).T
# → shape: (N, 3)  各行が (X, Y, Z) [mm単位]

# CSV出力
df = pd.DataFrame(points3D, columns=["X_mm", "Y_mm", "Z_mm"])
df.to_csv("points3D.csv", index=False)
```

## 注意事項：必要な画像枚数

精度の良いキャリブレーションには、様々な角度・位置でのチェスボード撮影が必要。

| 枚数（ペア） | 状況 |
|---|---|
| 2〜3枚 | 動作確認・概算値には可能 |
| 10〜20枚 | 実用レベルの精度 |
| 20枚以上 | 高精度が必要な場合 |

**撮影のポイント：**
- 両カメラで同時に同じチェスボードを写す
- チェスボードを様々な角度（傾き・回転）に変えて複数ペア撮影する
- 画面の中央・隅など異なる位置にボードを配置する

---

## 実装順序（推奨）

1. 画像確認・コーナー検出テスト（2枚でまず動作確認）
2. 単体キャリブレーション（各カメラのK・Dを推定）
3. ステレオキャリブレーション（R・Tを推定）
4. 平行化と可視化（結果の確認）
5. 追加撮影の必要性を判断（精度確認後）
6. 三角測量で3D座標を取得
