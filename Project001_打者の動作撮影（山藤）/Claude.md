# AIへの指示書

このドキュメントは、AIアシスタントに対する基本設定および指示をまとめたものです。AIは以下のルールを遵守して動作してください。

## 1. 基本的な出力ルール

- **言語設定**: ユーザーへの出力は、必ず**日本語**で行うこと。たとえユーザーからの入力が**英語**であったと
  しても、出力は**日本語**で行うこと。
- **エラー対応**: エラーログが入力された場合やエラーに遭遇した場合は、以下の2点を含めてわかりやすく日本語で出力すること。

  1. エラーの詳しい説明（原因の分析）
  2. 具体的な修正案・解決策の提示

## ユーザー

このプロジェクトでは、Github で共有しながら複数のユーザー（学生と教員、など）が並行して開発します。
ログを取る際は、ユーザーが誰なのかを、userinfo.md から取得して、記載してください。
userinfo.md は git 管理していません。ローカルのプロジェクトフォルダ内に userinfo.md が存在しない場合は、作成を促してください。

ユーザーロールが、student の場合は、AIは、コードの修正の際に、修正案を提示するだけにして、ファイルの更新を行わないこと。
student の場合は、エラーの原因や、アルゴリズムについて、詳しく説明すること。

## 2. ログ管理について

AIエージェントとユーザーとのやり取りのログは、logging_policy.md に従って記録・保存すること。
