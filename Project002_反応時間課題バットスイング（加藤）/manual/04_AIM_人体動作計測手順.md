## 前提条件

- Qualisys カメラシステムが起動・キャリブレーション済みであること
- QTM（Qualisys Track Manager）が起動できる状態であること
- NI DAQ ボードが PC に接続されていること
- MATLAB のカレントフォルダが `sandbox/1.2_AIM_人体動作計測` になっていること

## 手順チェックリスト

### 【準備】マーカーの貼付

- [ ] 上記マーカーセットの10点を解剖学的ランドマークに貼付する
- [ ] 反射面が上向き（カメラ方向）になっていることを確認する
- [ ] 左右対称に貼付されていることを確認する

---

### 【ステップ1】QTM 起動・プレビュー開始

- [ ] QTM を起動し、対象プロジェクトを開く
- [ ] **Capture → Preview** でプレビューを開始する

---

### 【ステップ2】3D ビューでマーカーを確認

- [ ] **View → 3D** に切り替える
- [ ] 10点すべてのマーカーが白い点として安定表示されていることを確認する
- [ ] 点滅・消えているマーカーがあれば立ち位置・マーカー向きを調整する

---

### 【ステップ3】キャプチャ設定を確認する

- [ ] **Project → Project Options → Capture** を開く
- [ ] フレームレート：100 Hz、計測時間：5〜10秒 になっていることを確認する

---

### 【ステップ4】QTM をトリガ待機状態にする

- [ ] **Capture → Capture** を選択する
- [ ] **Start** ボタンを押す
- [ ] ステータスバーに **"Waiting for trigger"** と表示されることを確認する

---

### 【ステップ5】MATLAB からトリガ送信・キャプチャ実施

- [ ] `2026-0421_01_practice_AIM_capture.m` を実行する
- [ ] コマンドウィンドウの案内に従い **Enter** を押す
- [ ] カウントダウン（5秒）の間に計測ポジションへ移動・静止する
- [ ] QTM のキャプチャが自動で開始されることを確認する
- [ ] 計測終了後、QTM 画面でキャプチャが記録されていることを確認する
- [ ] 2回目以降は QTM で再度 **Capture → Start** を押してから Enter を押す

---

### 【ステップ6】Trajectory Editor でラベリング

- [ ] **View → Trajectory Editor** を開く
- [ ] タイムライン上に10本のトラジェクトリが表示されることを確認する
- [ ] 各トラジェクトリを右クリック → **Label** でラベルを入力・Enter で確定する
- [ ] 全10点にラベルがつけられたことを確認する

---

### 【ステップ7】ギャップの確認と補間

- [ ] タイムライン上でトラジェクトリの途切れ（ギャップ）がないか確認する
- [ ] ギャップがある場合：右クリック → **Fill Gap** → **Polynomial** → OK で補間する

---

### 【ステップ8】3D 再生確認

- [ ] **View → 3D** に切り替えて再生ボタンを押す
- [ ] 全マーカーが滑らかに・連続して動いていることを確認する
- [ ] 不自然なジャンプ・スパイクがないことを確認する

---

### 【ステップ9】データのエクスポート（必要な場合）

- [ ] **File → Export** から形式を選んでエクスポートする
  - MATLAB で解析する場合：`.c3d` または `.mat`
  - テキストで確認する場合：`.tsv` / `.csv`

## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| カウントダウンが表示されない | カレントフォルダが `1.2_AIM_人体動作計測` か確認する |
| マーカーがプレビューに映らない | 立ち位置・マーカー向きを調整する |
| ギャップがある | カメラ配置を変更する、または Fill Gap で補間する |
| ラベルが保存されない | Enter キーで確定してから保存する |
| "Waiting for trigger" が出ない | QTM で Start ボタンを押してから Enter を押す |

詳細は [技術説明.md](../sandbox/1.2_AIM_人体動作計測/技術説明.md) を参照。
