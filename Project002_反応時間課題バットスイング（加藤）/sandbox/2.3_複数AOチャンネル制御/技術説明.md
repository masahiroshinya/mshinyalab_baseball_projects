# 2.3. 複数のAOチャンネルの出力制御

## 概要

このディレクトリでは、MATLABの **Data Acquisition Toolbox** を使用して、National Instruments (NI) 製のDAQボードから複数のアナログ出力（AO）チャンネルを同時に制御する方法について技術的な説明と検証を行います。複数のLEDや外部機器にパラレルに電圧を出力するための手順です。

## 基本的なプログラム手順 (MATLAB)

複数のチャンネルを使用する場合も、基本的な流れは1チャンネルの場合（2.1参照）と同じですが、チャンネルの追加方法と出力するデータの形式が異なります。

### 1. DataAcquisition オブジェクトの作成

```matlab
dq = daq("ni");
```

### 2. 複数の出力チャンネルの追加

`addoutput` 関数を呼び出して、必要な数だけチャンネルを追加します。

```matlab
% デバイスIDが"Dev1"の場合
% ao0 と ao1 の2つのアナログ出力チャンネルを追加
addoutput(dq, "Dev1", "ao0", "Voltage");
addoutput(dq, "Dev1", "ao1", "Voltage");
```

追加した順序が、後でデータを出力する際の列の順序になります（1列目がao0、2列目がao1）。

### 3. 電圧出力 (複数チャンネル同時)

`write` 関数を用いて電圧を出力します。複数のチャンネルに出力する場合、チャンネル数と同じ列数を持つ配列（またはテーブルやタイムテーブル）を渡します。

**例: ao0に5V、ao1に3.3Vを同時に出力する**

```matlab
% [ao0の出力値, ao1の出力値] のように横ベクトルで指定
write(dq, [5.0, 3.3]);
```

**例: 両方のチャンネルを同時に消灯（0V）する**

```matlab
write(dq, [0, 0]);
```

## 配線について

* `ao0`ピンとGNDの間にLED1（+抵抗）を接続します。
* `ao1`ピンとGNDの間にLED2（+抵抗）を接続します。
* 複数の機器をつなぐ場合、GNDはDAQ上の共通のGNDピン（または複数あるGNDピン）を利用して各回路を閉じるようにします。
