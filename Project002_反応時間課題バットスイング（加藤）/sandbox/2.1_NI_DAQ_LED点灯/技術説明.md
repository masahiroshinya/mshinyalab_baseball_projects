# 2.1. NI DAQボードにLEDを繋ぎ、MATLAB data acquisition toolboxを用いて電圧出力を行い、LEDを点灯させる

## 概要
このディレクトリでは、MATLABの **Data Acquisition Toolbox** を使用して、National Instruments (NI) 製のDAQボードから電圧出力を行い、接続されたLEDを点灯させるための基本的な技術・スクリプトを検証・記録します。

## 必要なハードウェア・ソフトウェア
*   **ハードウェア**: 
    *   NI DAQボード
    *   LED
    *   抵抗（LEDの過電流保護用）
    *   ブレッドボードおよびジャンパーワイヤ
*   **ソフトウェア**:
    *   MATLAB（Data Acquisition Toolbox がインストールされていること）
    *   NI-DAQmx ドライバ（NIデバイス用サポートパッケージ）

## 回路の基本構成
1. DAQボードの出力ピン（Analog Output `ao` または Digital Output `port/line`）とLEDのアノード（長い足）を接続します。
2. LEDのカソード（短い足）から抵抗を介して、DAQボードのGND端子に接続します。

---

## 基本的なプログラム手順 (MATLAB)

### 1. デバイスの認識確認
MATLABがNI DAQボードを認識しているか確認します。
```matlab
devices = daqlist;
disp(devices);
```
※ ここで表示される `ID` (例: `"Dev1"`) を以降のコードで使用します。

### 2. DataAcquisition オブジェクトの作成
Data Acquisition Toolboxの新しいインターフェース (`daq`) を使用してオブジェクトを作成します。
```matlab
dq = daq("ni");
```

### 3. 出力チャンネルの追加
チャンネルを追加して、電圧出力（Voltage）の設定を行います。
```matlab
% 例: デバイスIDが"Dev1"で、アナログ出力チャンネル0 ("ao0") を使用する場合
addoutput(dq, "Dev1", "ao0", "Voltage");
```

### 4. 電圧出力 (LED点灯・消灯)
`write` 関数を用いて電圧を出力します。アナログ出力では0〜5Vの範囲などで任意の電圧を指定できます。
※実際のテストスクリプトのコードは [test_led.m](test_led.m) を参照してください。

**LEDを点灯させる (例: 5V出力)**
```matlab
write(dq, 5);
```

**LEDを消灯させる (0V出力)**
```matlab
write(dq, 0);
```

## トラブルシューティング: デバイスが認識されない場合

もし `daqlist` を実行して以下のようなエラーが出た場合の対処法です。
> `Unable to detect 'ni' hardware: ...`

1. **Windows側での認識確認（手順の切り分け）**:
   エラーの原因がOS（ドライバ）かMATLAB側のパッケージかを切り分けるために、NIが提供する **「NI MAX (Measurement & Automation Explorer)」** ソフトを開き、「デバイスとインターフェイス」タブに目的のハードウェア（例: NI USB-6212 (BNC) "Dev1"など）が正常に表示されているか確認します。表示されていれば、OSとドライバは正常に機能しています。

2. **MATLABサポートパッケージのインストール**:
   NI MAXで認識されているにも関わらずMATLABで認識されない場合は、MATLABにNIデバイス用のサポートパッケージが入っていません。
   MATLABの機能リボンから **「アドオンの取得 (Get Add-Ons)」** を開き、
   `Data Acquisition Toolbox Support Package for National Instruments NI-DAQmx Devices`
   と検索して、サポートパッケージをインストールしてください。インストール後、MATLABから認識されるようになります（場合によってはPCの再起動が必要です）。
