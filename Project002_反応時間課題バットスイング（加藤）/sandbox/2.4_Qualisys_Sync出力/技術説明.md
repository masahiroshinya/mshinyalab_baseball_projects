# 2.4. Qualisys の sync チャンネルに伝える電圧を出力する

## 概要
このセクションでは、Qualisys（モーションキャプチャシステム）の計測開始タイミング等を制御・同期させるために、DAQボードから **Sync信号（トリガパルス）** を出力する方法を検証します。
Qualisysを含む多くの外部機器は、TTLレベル（0V ⇔ 5V）の電圧の立ち上がり（Edge）などをトリガとして認識します。

ここでは先ほどの「複数チャンネル出力」の応用として、LED（ao0, ao1）を点灯させつつ、それと同時に **QualisysのSync用チャンネル（ao2）に5Vの信号を一瞬だけ送る（矩形波パルスを送る）** プログラムを作成します。

## 配線の考え方
1. `ao0` と `GND`：LED 1
2. `ao1` と `GND`：LED 2
3. **`ao2` と `GND`：Qualisys の Sync In 端子へ接続**
   * 市販のBNCケーブルなどを介して、`ao2` の信号を Qualisysカメラシステム（Oqus等を束ねるユニット）の Sync In に入力します。
   * *(※ 現段階ではまだQualisysに繋がず、代わりにテスターなどで5Vが出ているか、もしくは別のLEDを繋いで一瞬だけ光るかを確認する形でもテスト可能です。)*

## パルスの送信方法について
Qualisysに「ここから開始だ」と伝えるためには、0Vを持続している状態から一瞬だけ5Vを出力し（パルスの立ち上がりエッジを作り）、ふたたび0Vに戻す必要があります。

```matlab
% 1. 初期状態（0V）
write(dq, [0, 0, 0]);

% 2. LED点灯と同時に、Sync(ao2)に5Vのトリガ信号を送信（立ち上がり）
write(dq, [5, 0, 5]); 

% 3. 一瞬だけ待って、トリガ信号を落とす（Pulse幅の確保）
pause(0.01); % 10ミリ秒（余裕を持たせた長さ）
write(dq, [5, 0, 0]); % LED(ao0)は点灯したまま、Sync(ao2)を0に戻す
```
上記のようにすることで、確実なトリガエッジをQualisysに認識させることができます。
