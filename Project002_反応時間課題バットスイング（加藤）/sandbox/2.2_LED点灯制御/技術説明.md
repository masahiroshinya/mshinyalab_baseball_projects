# 2.2. LEDの点灯のタイミングや時間を制御するプログラムを作成する

## 概要

このディレクトリでは、2.1で構築したNI DAQ + MATLABの環境を前提として、LEDの**点灯タイミング**や**点灯時間（継続時間）**を、ハードウェアクロックによって高精度に制御するプログラムを作成・検証します。

### ソフトウェアタイミング vs ハードウェアタイミング

| 方式                             | 代表的な関数                 | タイミング制御の主体                      | 精度の目安                   |
| -------------------------------- | ---------------------------- | ----------------------------------------- | ---------------------------- |
| ソフトウェアタイミング           | `pause()`, `tic`/`toc` | OS のタスクスケジューラ                   | ±数十ms（Windowsでは~15ms） |
| **ハードウェアタイミング** | `preload()` + `start()`  | **DAQボードのハードウェアクロック** | **±数十μs以下**      |

反応時間実験ではmsオーダーの精度が求められます。`pause()` などのソフトウェアタイミングはOSのスケジューラに依存するため、特にWindowsでは誤差が大きく不適切です。

**正しいアプローチ**: あらかじめ出力したい電圧の時系列（波形データ）をMATLAB上でベクトルとして生成し、`preload()` でDAQのハードウェアバッファに書き込んでから `start()` で出力を開始します。実際の電圧出力タイミングはDAQボード上のクロックが制御するため、OSの影響を受けません。

---

## 基本的な仕組み

### サンプリングレートと時間分解能

DAQオブジェクトのサンプリングレート (`dq.Rate`) を設定すると、ハードウェアはそのレートで1サンプルずつ順番に電圧を出力します。

- `dq.Rate = 1000` → 1ms ごとに1サンプル出力（1ms分解能）
- `dq.Rate = 10000` → 0.1ms ごとに1サンプル出力（0.1ms分解能）

波形ベクトルの各要素が、それぞれのタイムステップに対応した出力電圧になります。

### 波形の生成イメージ

例として「500ms点灯 → 1000ms消灯」の波形を `dq.Rate = 1000` で生成する場合：

```
インデックス:  1~500      501~1500
電圧:         5.0V  ...  0.0V  ...
時刻:          0~499ms   500~1499ms
```

これを `preload()` でDAQバッファに渡し、`start()` で出力します。

## プログラム例

### 例1: 一定時間だけ点灯させる

```matlab
% --- パラメータ ---
sample_rate  = 10000;   % サンプリングレート [Hz]（0.1ms分解能）
on_duration  = 0.5;     % 点灯時間 [秒]
off_duration = 1.0;     % 消灯時間 [秒]（波形終端を0Vにするためのパディング）
voltage_on   = 5.0;     % 点灯電圧 [V]
voltage_off  = 0.0;     % 消灯電圧 [V]

% --- DAQの初期化 ---
dq = daq("ni");
dq.Rate = sample_rate;
addoutput(dq, "Dev1", "ao0", "Voltage");

% --- 波形ベクトルの生成 ---
n_on  = round(on_duration  * sample_rate);  % 点灯サンプル数
n_off = round(off_duration * sample_rate);  % 消灯サンプル数

waveform = [voltage_on  * ones(n_on,  1);
            voltage_off * ones(n_off, 1)];

% --- DAQバッファへの書き込みと出力開始 ---
preload(dq, waveform);
start(dq, "Finite");    % 波形を全出力して自動停止
```

---

### 例2: 繰り返し点滅させる（全波形を事前生成）

試行数分の波形を連結して、まとめてDAQに渡します。

```matlab
% --- パラメータ ---
sample_rate  = 10000;
n_trials     = 5;
on_duration  = 0.3;     % 点灯時間 [秒]
isi          = 1.5;     % 刺激間隔（点灯開始から次の点灯開始まで）[秒]
voltage_on   = 5.0;
voltage_off  = 0.0;

% --- DAQの初期化 ---
dq = daq("ni");
dq.Rate = sample_rate;
addoutput(dq, "Dev1", "ao0", "Voltage");

% --- 波形ベクトルの生成（全試行分を連結） ---
n_on  = round(on_duration * sample_rate);
n_off = round((isi - on_duration) * sample_rate);  % 1試行あたりの消灯サンプル数

single_trial = [voltage_on  * ones(n_on,  1);
                voltage_off * ones(n_off, 1)];

waveform = repmat(single_trial, n_trials, 1);

% --- DAQバッファへの書き込みと出力開始 ---
preload(dq, waveform);
start(dq, "Finite");
```

---

### 例3: ランダムな遅延（ジッター）をもたせた点灯

各試行の消灯時間をランダムに変えることで、刺激呈示タイミングにジッターを加えます。
波形を1試行ずつ生成・連結してから、まとめてDAQに渡します。

```matlab
% --- パラメータ ---
sample_rate  = 10000;
n_trials     = 5;
on_duration  = 0.3;       % 点灯時間 [秒]
jitter_min   = 1.0;       % ISIの最小値 [秒]
jitter_max   = 3.0;       % ISIの最大値 [秒]
voltage_on   = 5.0;
voltage_off  = 0.0;

% --- DAQの初期化 ---
dq = daq("ni");
dq.Rate = sample_rate;
addoutput(dq, "Dev1", "ao0", "Voltage");

% --- 波形ベクトルの生成（各試行でISIをランダム化） ---
n_on   = round(on_duration * sample_rate);
waveform = [];

for i = 1:n_trials
    isi_i  = jitter_min + (jitter_max - jitter_min) * rand();  % ランダムISI
    n_off  = round((isi_i - on_duration) * sample_rate);

    fprintf('試行 %d: ISI = %.3f 秒\n', i, isi_i);

    trial_wave = [voltage_on  * ones(n_on,  1);
                  voltage_off * ones(n_off, 1)];
    waveform = [waveform; trial_wave];
end

% --- DAQバッファへの書き込みと出力開始 ---
preload(dq, waveform);
start(dq, "Finite");
```

## 重要な関数のまとめ

| 関数                      | 役割                                                               |
| ------------------------- | ------------------------------------------------------------------ |
| `dq.Rate = N`           | ハードウェアサンプリングレート [Hz] の設定                         |
| `preload(dq, waveform)` | 波形データをDAQのハードウェアバッファに書き込む                    |
| `start(dq, "Finite")`   | ハードウェアクロックで波形の出力を開始（終端まで出力して自動停止） |
| `stop(dq)`              | 出力を中断する場合                                                 |

## トラブルシューティング

### `start()` の引数エラー：`'Duration'` は無効

**エラーメッセージ：**

```
次を使用中のエラー: daq.interfaces.DataAcquisition/start (行 469)
'Mode' の値は無効です。入力は次の値のいずれか 1 つと一致する必要があります。
'Finite', 'Continuous', 'RepeatOutput'
入力 'Duration' は有効な値のいずれとも一致しません。
```

**原因：**
`start()` の第2引数に `"Duration"` を渡したが、このような引数は存在しない。

**解決策：**
波形を1回出力して自動停止する場合は `"Finite"` を使用する。

```matlab
% 誤り
start(dq, "Duration");

% 正しい
start(dq, "Finite");
```

有効なモードの一覧：

| モード             | 動作                       |
| ------------------ | -------------------------- |
| `"Finite"`       | 波形を1回出力して自動停止  |
| `"Continuous"`   | 継続的に出力し続ける       |
| `"RepeatOutput"` | 波形を繰り返し出力し続ける |
